Modified eCommerce Website of Watches

Comparative Analysis of three e-commerce Websites: Mk, Rolex, Bvlgari and our website
Timely Treasure.

Introduction

With the increasing trend of online shopping,
e-commerce and fashion watches have become
significant for both clients and brands. This research
aims to evaluate branded watch e-commerce platforms by
analyzing usability, product demonstration, customer reviews,
and shopping experience. Each site is assessed based on:

  • Visual Design
  • User Interface (UI)
  • User Experience (UX)
  • Technology Used
  • Navigation Structure
  • Features and Functionality

Website 1: Rolex

Visual Design: Minimalist with high-quality images and a modern color scheme.

User Interface: Interactive with hover effects and clearly labeled buttons.

User Experience: Smooth browsing, quick load times, and responsive filters.

Technology Used: HTML, CSS, JavaScript, React.

Navigation Structure: Sticky navigation bar.

Features: Live chat, detailed filters, and a watch comparison tool.

Website 2: Bulgari

Visual Design: Luxurious typography with full-screen images.

User Interface: Elegant and refined with smooth animations.

User Experience: Minimalist design for a seamless shopping experience.

Technology Used: HTML, CSS, JavaScript.

Navigation Structure: Tabbed menu with minimal distractions.

Features: Personalized recommendations and 360-degree product views.

Website 3: MK

Visual Design: Bold and vibrant with youthful appeal.

User Interface: Clear and functional with dropdown menus.

User Experience: Quick checkout and easy customization.

Technology Used: HTML, CSS, JavaScript, Angular.

Navigation Structure: Dropdown menu organizing products by brand and price.

Features: Custom watch builder, wish list, and loyalty rewards.

Website 3: Timely Treasure

Visual Design: Simple typography, and minimal styling.

User Interface: product cards, a navigation bar, a search bar, and a popup modal.

User Experience: Straight fordward and functional.

Technology Used: HTML, CSS, JavaScript.

Navigation Structure: Fixed navigation bar

Features: Hot label on watch and functional search bar
